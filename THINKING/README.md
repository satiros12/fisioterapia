# Fisiotherapy Project Documentation

## Project Overview
This repository implements a comprehensive physiotherapy management system designed to track patient progress, generate treatment plans, and facilitate clinical documentation. The system includes:

- Patient record management
- Exercise prescription and tracking
- Progress monitoring with visual reports
- Clinical documentation and audit trails

## Key Features
- Patient profile management with medical history
- Exercise library with descriptions and images
- Treatment plan creation with sequential progression
- Progress tracking using EVA (Evaluación del Dolor) scoring
- Integration with visual resources (images, diagrams)

## Technology Stack
- Python 3.12.3
- Python package management with UV
- Flask (backend framework)
- SQLite (lightweight database for prototyping)
- Bootstrap 5 (front-end UI components)
