# PLAN DE FISIOTERAPIA

Se basa en esta página, pero quisiera una para mi o una app :)
https://ejercicios.sermef.es/#/

La mayoría de tratamientos en la fisioterapia se basan en protocolos que emplean como pauta fundamental el ejercicio terapéutico.

Los ejercicios que se realizan en el tratamiento de cada patología, aunque se realizan a criterio del profesional, hay varios protocolos que se usan para simplificar la ejecución del tratamiento.

(No sé cómo sería una clasificación adecuada pero podría ser tipo)
Clasificación por segmento corporal y articulación.

Quiero poder tener un sistema en el que al abrir el segmento corporal, por ejemplo: columna, se despliegue en cervical, lumbar, torácica, sacra.

Y en cada zona tener los ejercicios de estiramientos, movilidad activa, fuerza.

Y yo como fisioterapeuta poder elegir los ítems indicados para cada lesión.

Ejemplo: epicondilitis. Y que se despliegue todos los ejercicios del tratamiento y yo pueda ir seleccionando el que me parezca más adecuado para cada etapa de evolución de la patología.

El punto es poder explicar de forma detallada cada ejercicio con imágenes para que el paciente siga las instrucciones y vaya haciendo check en la actividad.

### COLUMNA
- CERVICAL / CUELLO
  - **Nivel 1**
    - Ejercicios de estiramientos: Flex, extensión, rotación izquierda, rotación derecha, inclinación izquierda, inclinación derecha.
    - Hacer check de dolor con EVA (Escala Visual Analógica). Si el EVA es > 6 en cualquier ejercicio, no se permite avanzar al siguiente nivel.
    - Al alcanzar el ejercicio sin dolor, se desbloquea el siguiente ejercicio (nivel 2).
    - **Nivel 2**
      - Estiramientos combinados: movimientos flexión‑rotación, extensión‑rotación, etc.
      - Hacer check de dolor con EVA (EVA > 6 impide avanzar).
      - **Movilidad activa 2**: imágenes y buenas instrucciones. Check.
      - **Fuerza isométrica**: imagen indicación + imagen de musculatura activa. Check.
      - **Fuerza excéntrica y aumento progresivo**: imágenes. Check. Adjuntar calificación EVA para el dolor comparativa.
  - **Nivel 2** (continúa con progresiones similares)
    - ...

### CLAVIJERAS DE USO
- Al abrir un segmento (ej. columna), se visualiza un ** atlas anatómico** que muestra la zona a tratar.
- Se pueden filtrar los ejercicios por **patología** o por **músculo** si se desea una navegación más específica.
- Cada ejercicio incluye:
  - Imagen de referencia.
  - Instrucciones paso a paso.
  - campo de check/confirmación.
  - registro de EVA para comparar el dolor antes y después del ejercicio.

Este diseño permitirá que tanto el fisioterapeuta como el paciente tengan una guía clara, visual y estructurada para avanzar en la rehabilitación de forma controlada y personalizada.
