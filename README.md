# FisioTherapy

A Python-based physiotherapy application for tracking patient progress and analyzing rehabilitation data.

## Features
- Patient progress tracking and milestone monitoring
- Data analysis with numpy and pandas
- Report generation for clinical documentation
- Modular architecture for extensibility

## Installation

1. Clone the repository
```bash
git clone https://github.com/satiros12/fisioterapia.git
cd fisioterapia/src
```

2. Install dependencies
```bash
uv pip install -r requirements.txt
```

## Usage
Run the main application:
```bash
python -m src.main
```
